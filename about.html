Questions

Questions Answers
1. What does the server in client server architecture do? The server processes requests from the client and returns the corresponding data.
2. What does the client do? The client sends requests to the server and outputs or shows the data it gets from the server
3. What is an IP address? An address used to identify a device on the internet or a local network
4. What is DNS? DNS is the Domain Name System which routes your web browser to websites through domains.
5. What is the difference between Internet and WWW? The internet is the network of connected computers that is used to connect devices globally and WWW or World Wide Web is the pages and websites that are on the internet
6. Explain what HTML is? HTML is Hyper Text Markup Language which is the standard language used to create and describe the structure of webpages. The HTML tells the browser how to display the content of the page to the device its being displayed on.
